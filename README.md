## Purpose

To practice using web APIs (coordinates, size, etc.)

## Skills

- HTML + CSS
- JavaScript
- Web APIs

## Features

- On mouse move, update a pin's position and its coordinates.
- On mouse click, fix a pin on the clicked position.
- On page load, initiate a pin's position and its coordinates in the center.
