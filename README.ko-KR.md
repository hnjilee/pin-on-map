## 프로젝트의 목적

Web APIs (좌표, 사이즈 등) 활용 연습

## 사용기술

- HTML + CSS
- JavaScript
- Web APIs

## 주요기능

- 마우스 이동 시, 핀의 위치와 좌표값 업데이트
- 마우스 클릭 시, 핀을 클릭된 위치에 고정
- 페이지 로딩 시, 핀의 위치와 좌표값 중앙으로 초기화
